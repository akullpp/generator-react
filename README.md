# Generator for React Projects

## Usage

```sh
git clone https://github.com/akullpp/generator-react.git
cd generator-react
npm link
yo react
```

## Stack

### Production

- axios
- create-react-app
- material-ui with icons
- react-dom
- react-router
- styled-components

### Development

- concurrently
- editorconfig
- eslint with Airbnb
- husky
- prettier
- react-testing-library
